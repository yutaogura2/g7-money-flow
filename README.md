# G7一次情報＋世界のマネーの流れ ダッシュボード

G7（主要7カ国）の財務相・中央銀行総裁会議／首脳会議の公式一次情報を日本語に翻訳・考察し、さらに**世界のマネーの流れ（マネーサプライ・中銀バランスシート・ネット流動性）と市場・リスク指標（金利スプレッド・クレジット・VIX・為替・CFTC建玉）**を、Zスコア/シグナル付きで一望できるダッシュボードです。

**🌐 ライブデモ（ブラウザでそのまま操作）**: https://yutaogura2.github.io/g7-money-flow/

> 投資助言・売買推奨ではありません。考察は公式一次データに基づく参考情報です。

## 主な機能
- 7タブ：概要(週次ブリーフィング) / 会議・声明ログ / イベントカレンダー / 考察ビュー / マクロ / マネーの流れ / 市場・リスク
- データ源：G7一次情報＋FRED・ECB・日銀・CFTC・合成（マクロ45系列）。すべて公式一次データ。
- シグナル：全指標にZスコア・パーセンタイル・信号色、リスクレジーム総合、直近ムーバー、データ鮮度警告。
- 週次自動更新（Windowsタスク）＋全履歴をExcel/Gitで保持。

## 使い方
1. `index.html` をダブルクリックしてブラウザで開く。
2. タブで「概要 / 会議・声明ログ / イベントカレンダー / 考察ビュー」を切り替え。
3. 過去ログ全体は `G7_一次情報ログ.xlsx`（3シート）で確認・フィルタできます。

> 注意: 一部ブラウザ（Chrome等）はローカルファイル（`file://`）の制約が厳しい場合があります。
> その場合は、このフォルダで `python -m http.server 8765` を実行し、ブラウザで
> `http://localhost:8765/` を開いてください。

## 更新（情報の追加）
Claude に「G7情報を更新して」と頼んでください。Claude が公式ソースを巡回し、
`data.json` に追記して `python build.py` を実行 → ダッシュボードとExcelに反映されます。
詳細は `更新手順.md` を参照。

## ファイル
- `data.json` … データ本体（唯一の編集対象）
- `build.py` … data.json → `data.js` + `G7_一次情報ログ.xlsx` を生成
- `index.html` … ダッシュボード
- `G7_一次情報ログ.xlsx` … Excel全archive（自動生成）
- `更新手順.md` … 更新手順・対象公式ソース一覧
- `tests/` … build.py のテスト（`python -m pytest`）

## セットアップ（初回のみ）

```
python -m pip install -r requirements.txt
```
