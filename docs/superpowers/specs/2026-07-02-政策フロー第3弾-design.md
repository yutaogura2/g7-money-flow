## 4. A1 政策サイクル（build.py）

- `build_policy_cycle(series) -> list`：payloadの `indicator=="政策金利"` の各系列について、
  - `prior` = history から「latest_date の183日前以前の最新値」（`[[d,v],...]` を走査）。
  - `delta6m = latest - prior`（round 2）。prior無しは skip。
  - `dir`: `delta6m > 0.1`→`up`(引き締め)、`< -0.1`→`down`(緩和)、間→`flat`(据置)。
  - 要素: `{country, latest, delta6m, dir}`。
- `build_macro_payload` 内で `signals["policy_cycle"] = build_policy_cycle(series)` を付与（build_signals の戻りに追記）。

## 5. A2 イベントマーカー（index.html）

- `lineChartSVG(seriesList, opts)` に `opts.markers`（ISO日付配列）を追加：チャートのx範囲内のマーカーごとに、**最も近いx位置に縦の破線**（stroke `#8a6d3b` 相当の控えめ色, dasharray）を描画。範囲外は無視。
- 適用: 「マネーの流れ」の世界流動性チャートと週次フローチャートに、
  - `DATA.calendar` の `category ∈ {FOMC, 日銀, ECB, G7首脳, G7財務}` の日付、
  - `MOF_INTERVENTION` の history 日付（介入実施月）
  を渡す。カード下に注記「縦線＝政策イベント・為替介入」。

## 6. 政策サイクルカード（index.html）

- 表示形式: `米国 3.63% ↓緩和 ／ ユーロ圏 2.25% →据置 ／ 日本 … ／ 英国 …`（dirで色: up=赤系, down=緑系, flat=グレー）。
