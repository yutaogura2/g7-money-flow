## 1. 目的

G7（主要7カ国）の **財務大臣・中央銀行総裁会議** および **首脳会議（サミット）** で決まったこと・発表された公式な一次情報（コミュニケ／共同声明等）を収集し、日本語に翻訳して整理し、**投資判断のための考察** を付けてまとめるツール。

- 外国語の一次情報は日本語に翻訳する（日本語を主、原文も保持）。
- 過去ログを **Excel** に全archiveとして残す。
- **イベントカレンダー**（G7＋主要マクロイベント）も提供する。

> 用途は投資判断の参考。考察は投資助言・売買推奨ではない。

## 3. 採用アーキテクチャ（A案：ローカル静的＋Pythonビルド）

```
[私(Claude)が公式ソースを巡回]
   └─ 翻訳・要約・考察 → data.json に追記
          └─ build.py 実行
                ├─→ data.js                （index.html が読む）
                └─→ G7_一次情報ログ.xlsx     （過去ログ全archive）
[ユーザー] index.html をブラウザで開く / Excelを見る
```

- **single source of truth は `data.json` 1つ**。ブラウザ表示もExcelも常にここから生成するため二重管理・データのズレが起きない。
- 完全ローカル・オフライン。OneDrive上のこのフォルダ内で完結。
- `index.html` は `data.js` を `<script src>` で読み込む（`file://` でも動く＝Chromeのfetch/CORS制約を回避）。

### 採用理由（他案との比較）
- **B案（データ埋め込み単一HTML）**：ファイル1個で済むが、データと表示が密結合で履歴差分・保守がしづらい。Excelも別管理になる。→ 不採用。
- **C案（ホスティング型Webアプリ）**：常時オンライン・共有可だが、デプロイ・APIキー・運用コストが必要で個人用途にはオーバースペック。→ 不採用。

## 6. ダッシュボード（index.html）画面設計

- **概要（トップ）**：最終更新日／直近の予定イベント数件／最新の会議ログ数件／注目タグ。
- **会議・声明ログ**：日付・種別・議長国・タグでフィルタ。各カードを開くと〔要約〕〔決定事項（箇条書き）〕〔主要部分の全訳〕〔考察（4観点をタブ/見出しで）〕〔原文リンク〕。**日本語を主表示、原文（原題・原文抜粋）はトグルで開閉**。
- **イベントカレンダー**：時系列リスト（＋月別グルーピング）。カテゴリ別フィルタ、予定/済の色分け。関連会議ログへのリンク。
- **考察ビュー**：4観点（為替金利／商品エネルギー金／セクター／地政学）ごとに、該当する決定・発言を時系列で集約表示。
- 共通：画面下部に免責表示。日本語UI。レスポンシブ（PCブラウザ主、スマホでも崩れない程度）。

## 7. Excel（G7_一次情報ログ.xlsx）

- シート構成：
  - `会議ログ`：1行=1会議/声明。列＝日付・種別・議長国・邦題・原題・要約・決定事項・主要部分の全訳・考察(為替金利)・考察(商品)・考察(セクター)・考察(地政学)・方向性メモ・タグ・原文URL（ハイパーリンク）。
  - `カレンダー`：1行=1イベント。列＝日付・名称・カテゴリ・状態・関連会議ID・備考。
  - `考察サマリ`：観点×時系列のクロス整理（4観点ごとの主要論点を日付順に）。
- 見出し行は固定（ウィンドウ枠固定）・オートフィルタ有効。
- `build.py` 実行で毎回 data.json から全件再生成（＝常に全ログを反映）。

## 9. 更新フロー

1. ユーザーが「更新して」と依頼（または定期実行を設定）。
2. 私が対象公式ソースを巡回し、新規/更新の声明を取得。
3. 翻訳・要約・決定事項抽出・4観点の考察を作成し、`data.json` の `meetings`/`calendar` に追記。`meta.last_updated` を更新。
4. `build.py` を実行 → `data.js` と `G7_一次情報ログ.xlsx` を再生成。
5. ユーザーは `index.html` を再読込（または開く）／Excelを確認。

## 10. エラーハンドリング・運用上の注意

- 一次情報が確認できない/取得できない場合は、その旨を明記し、推測で埋めない（出典のない記載をしない）。
- 公式ソースのURL変更（議長国交代でサイトが変わる等）に備え、`更新手順.md` にソース一覧を保持し、取得時に都度検証する。
- 翻訳は原文の意味を保持。重要語（制裁・凍結・介入・コミュニケ等）は原語併記可。
- 考察は投資の参考情報であり助言ではない旨を、画面・Excel・READMEに明記。
