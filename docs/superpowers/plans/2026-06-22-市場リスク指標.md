**Goal:** プロ向け市場・リスク指標（FRED14系列＋合成のFRBネット流動性）を追加し、新タブ「市場・リスク」に集約する。既存「マクロ」はファンダメンタルのみ表示にする。

**Tech Stack:** Python 3.12（標準ライブラリ、既存 openpyxl）、Vanilla JS、pytest。新規依存なし。

## Global Constraints
- 配置ルート相対パス。UTF-8。値はFRED公式のみ。生成物は手編集しない。
- ネット流動性 = `WALCL − WTREGEN − RRPONTSYD×1000`（百万ドル）→ `/1e6`（兆ドル）。WALCL日付に整列、TGA/RRPは直近以前値。

def _nearest_prior(pairs: list, target: str):
    val = None
    for d, v in pairs:
        if d <= target:
            val = v
        else:
            break
    return val


def compute_netliq(walcl: list, tga: list, rrp: list) -> list:
    tga_s = sorted(tga)
    rrp_s = sorted(rrp)
    out = []
    for d, w in sorted(walcl):
        t = _nearest_prior(tga_s, d)
        r = _nearest_prior(rrp_s, d)
        if t is None or r is None:
            continue
        net = (w - t - r * 1000.0) / 1e6   # 百万ドル基準 → 兆ドル
        out.append((d, round(net, 4)))
    return out
```

function renderIndicatorGrid(viewId, series, selectId){
  const el = document.getElementById(viewId);
  if(!series.length){
    el.innerHTML = '<div class="card meta-row">データがありません。<code>python fetch_macro.py</code> を実行してください。</div>';
    return;
  }
  const inds = [...new Set(series.map(s => s.indicator))];
  const sections = inds.map(ind => `
    <div class="macro-section" data-ind="${esc(ind)}">
      <h3 style="margin:16px 0 8px; border-bottom:1px solid var(--line); padding-bottom:4px">${esc(ind)}</h3>
      <div style="display:grid; grid-template-columns:repeat(auto-fill,minmax(240px,1fr)); gap:12px">
        ${series.filter(s => s.indicator === ind).map(macroCard).join('')}
      </div>
    </div>`).join('');
  el.innerHTML = `
    <div class="card">
      <select id="${selectId}" style="padding:8px; background:var(--bg); color:var(--txt); border:1px solid var(--line); border-radius:6px;">
        <option value="">すべてのカテゴリ</option>
        ${inds.map(i => `<option>${esc(i)}</option>`).join('')}
      </select>
    </div>
    ${sections}`;
  document.getElementById(selectId).addEventListener('change', e => {
    const v = e.target.value;
    el.querySelectorAll('.macro-section').forEach(sec =>
      sec.style.display = (!v || sec.dataset.ind === v) ? '' : 'none');
  });
}

function renderMacro(){
  const all = (DATA.macro && DATA.macro.series) || [];
  renderIndicatorGrid('view-macro', all.filter(s => s.group !== 'market'), 'macro-ind');
}

function renderMarket(){
  const all = (DATA.macro && DATA.macro.series) || [];
  renderIndicatorGrid('view-market', all.filter(s => s.group === 'market'), 'market-ind');
}
```
