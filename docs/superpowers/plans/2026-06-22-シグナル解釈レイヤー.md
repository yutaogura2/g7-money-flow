**Goal:** 既存の全履歴から各指標のZスコア/パーセンタイル・信号色・リスクレジーム・直近ムーバーを算出し、ダッシュボードを「シグナルボード」化する（新規データ取得なし）。

**Tech Stack:** Python 3.12（標準ライブラリ）、Vanilla JS、pytest。新規依存なし。

## Global Constraints
- 配置ルート相対パス。UTF-8。新規データ取得なし（build のみ）。
- std は母標準偏差（÷N）。std==0 は z=0・delta_z=0 で安全に。
- 統計は **全期間**（CSV全行）から計算。表示用 history は従来どおり間引き。
- regimeラベル境界: `score>=0.5`→リスクオフ(warn)、`<=-0.5`→リスクオン(calm)、間→中立(ok)。
- 既存テストを壊さない。Gitはmain、各タスク末尾でコミット。
- 既存: `build._read_series_csv`/`build_macro_payload`/`macro_timeseries_rows`、`index.html` の `macroCard/renderOverview/esc/fmtYoy`。

```python
def _series_stats(values: list) -> dict:
    if not values:
        return {"mean": 0.0, "std": 0.0, "zscore": 0.0, "pctile": 0}
    latest = values[-1]
    n = len(values)
    mean = sum(values) / n
    std = (sum((v - mean) ** 2 for v in values) / n) ** 0.5
    z = round((latest - mean) / std, 2) if std else 0.0
    pct = round(sum(1 for v in values if v <= latest) / n * 100)
    return {"mean": mean, "std": std, "zscore": z, "pctile": pct}


def _value_on_or_before(rows: list, target: str):
    val = None
    for r in rows:
        if r[0] <= target:
            val = r[1]
        else:
            break
    return val
```

```python
def build_signals(series: list) -> dict:
    stresses = [s["stress"] for s in series if s.get("stress") is not None]
    if stresses:
        score = round(sum(stresses) / len(stresses), 2)
        if score >= 0.5:
            label, level = "リスクオフ", "warn"
        elif score <= -0.5:
            label, level = "リスクオン", "calm"
        else:
            label, level = "中立", "ok"
    else:
        score, label, level = None, "—", "ok"
    ranked = sorted(series, key=lambda s: abs(s.get("delta_z") or 0), reverse=True)[:6]
    movers = [{
        "id": m["id"], "country": m["country"], "indicator": m["indicator"],
        "latest": m["latest"], "delta": m.get("delta", 0), "delta_z": m.get("delta_z", 0),
        "dir": "up" if (m.get("delta") or 0) >= 0 else "down",
    } for m in ranked]
    return {"regime": {"score": score, "label": label, "level": level}, "movers": movers}
```

`build_macro_payload` の `return {"series": series}` を次に置換:

```javascript
function signalBadge(s){
  if(s.zscore == null) return '';
  let flag = '';
  if(s.stress != null){
    const st = s.stress;
    const c = st>=1.5 ? '#ff5a5a' : (st>=0.5 ? '#ffb74f' : (st<=-0.5 ? '#4fd08c' : '#9fb2cf'));
    const lbl = st>=1.5 ? '強ストレス' : (st>=0.5 ? 'ややストレス' : (st<=-0.5 ? '緩和' : '中立'));
    flag = ` ・ <span style="color:${c}">● ${lbl}</span>`;
  }
  return `<div class="meta-row">Zスコア ${esc(s.zscore)} ・ ${esc(s.pctile)}%ile${flag}</div>`;
}
```
